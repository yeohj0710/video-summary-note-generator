영상·음성 요약 노트 생성기 사용설명서

영상이나 오디오를 넣으면 원본 파일, 읽기 쉬운 전체 스크립트 TXT, 상세 요약 TXT를 같은 이름으로 만들어 주는 프로그램입니다.

developed by yeohj0710

2. 가장 빠른 사용 순서

  1. 영상·음성 요약 노트 생성기.exe를 실행합니다.
  2. 유튜브/릴스 링크를 넣거나, 내 컴퓨터의 영상/오디오 파일을 선택합니다.
  3. OpenAI API 키를 입력합니다.
  4. 저장 폴더가 생성된 노트인지 확인합니다.
  5. 필요하면 요약 설정에서 자동 요약 길이를 끄고 원하는 문장 수를 입력합니다.
  6. 노트 만들기를 누릅니다.
  7. 완료되면 결과 폴더 열기를 눌러 영상, 전체 스크립트, 요약 파일을 확인합니다.
  8. 진행 상황 로그에서 이번 작업의 예상 API 비용을 확인합니다.

3. TXT 파일 형식

일반 TXT에는 내부 처리용 시간 라벨이나 구간: 같은 문구를 넣지 않습니다. 전체 전사문만 들어갑니다.

문장은 읽기 편하도록 널널하게 줄바꿈됩니다.

첫 번째 문장입니다.

두 번째 문장입니다.

세 번째 문장입니다.

_요약.txt 파일은 전체 스크립트를 마지막에 한 번 더 읽고 만든 상세 요약본입니다. 수치, 날짜, 금액, 조건, 인물/회사/제품명, 중요한 예외와 결론은 최대한 살리고 반복 표현만 줄이는 방식으로 생성합니다.

요약은 기본적으로 전체 스크립트 문장 수의 약 1/5을 목표로 만들며, 프로그램의 요약 설정에서 직접 문장 수를 지정할 수도 있습니다.

4. 링크 영상

유튜브나 릴스 링크를 넣으면 영상이 먼저 다운로드되고, 같은 이름의 전체 스크립트 TXT와 요약 TXT가 만들어집니다. 모두 생성된 노트 폴더에 저장됩니다.

로그인이 필요한 릴스나 유튜브 영상은 브라우저 쿠키 사용 옵션을 켜야 할 수 있습니다.

5. 내 컴퓨터 영상/오디오

내 컴퓨터 파일을 선택하면 원본 영상이나 오디오가 생성된 노트 폴더에 복사되고, 같은 이름의 전체 스크립트 TXT와 요약 TXT가 만들어집니다.

mp3, wav, m4a, amr, caf, aiff 같은 오디오 파일만 넣어도 사용할 수 있습니다.

휴대폰 녹화 파일에서 자주 나오는 mov, 3gp, m4v 파일도 선택할 수 있습니다.

7. 비용과 보안

  • OpenAI API 사용량에 따라 비용이 발생할 수 있습니다.
  • 작업 완료 후 진행 상황 로그에 한화 기준 예상 API 비용이 표시됩니다. 실제 청구액은 OpenAI 가격 변경, 환율, 계정 조건에 따라 조금 달라질 수 있습니다.
  • API 키가 없다면 프로그램의 API 키 받는 법 버튼으로 HTML 가이드를 열 수 있습니다.
  • 링크와 파일 선택이 헷갈리면 프로그램의 가져오기 도움말 버튼을 누르세요.
  • 영상이 길수록 전사 시간과 비용이 늘어납니다.
  • 공용 PC에서는 API 키 저장 옵션을 권장하지 않습니다.
