영상·음성 요약 노트 생성기 사용설명서

영상이나 오디오를 넣으면 원본 파일, 전체 스크립트 TXT, 상세 요약 TXT를 같은 이름으로 만들어 주는 프로그램입니다.

developed by yeohj0710

1. 가장 빠른 사용 순서

  1. 영상·음성 요약 노트 생성기.exe를 실행합니다.
  2. 링크로 가져오기 또는 내 컴퓨터 파일 중 하나를 선택합니다.
  3. 유튜브/릴스 링크를 붙여넣거나, 영상/오디오 파일을 선택합니다.
  4. OpenAI API 키를 입력합니다.
  5. 저장 폴더는 기본값 그대로 두면 됩니다.
  6. 필요하면 요약 설정에서 요약 길이를 조정합니다.
  7. 노트 만들기를 누릅니다.
  8. 완료되면 결과 폴더 열기를 눌러 결과 파일을 확인합니다.

2. 영상 또는 오디오 넣기

링크로 가져오기

유튜브나 릴스 링크를 붙여넣으면 프로그램이 영상을 먼저 저장한 뒤 전체 스크립트와 요약을 만듭니다.

로그인이 필요한 릴스나 유튜브 영상은 브라우저 쿠키 사용 옵션이 필요할 수 있습니다.

내 컴퓨터 파일

이미 가지고 있는 영상이나 녹음 파일을 직접 선택할 수 있습니다. 선택한 파일은 결과 폴더에 복사됩니다.

mp4, mov, m4v, mp3, wav, m4a 같은 파일을 사용할 수 있습니다.

3. OpenAI API 키

전사와 요약을 만들기 위해 본인 OpenAI API 키가 필요합니다.

키가 없다면 프로그램 안의 API 키 받는 법 버튼을 누르면 자세한 안내가 열립니다.

기본 설정 기준 10분 영상은 보통 수십 원 정도입니다. 영상 길이, 말의 양, 문장 정리 옵션에 따라 달라질 수 있습니다.

작업이 끝나면 진행 상황 로그에 한화 기준 예상 비용이 표시됩니다.

4. 요약 설정

처음에는 자동으로 요약 길이 결정을 켜 둔 상태로 사용하는 것을 권장합니다.

요약이 너무 짧거나 길다고 느껴지면 자동 체크를 끄고 원하는 문장 수를 직접 입력할 수 있습니다.

전사문 맞춤법/띄어쓰기 정리 사용을 끄면 비용과 시간을 조금 아낄 수 있지만, 전사문이 덜 정돈될 수 있습니다.

5. 결과 파일

작업이 끝나면 생성된 노트 폴더 안에 같은 이름의 파일 3개가 생깁니다.

생성된 노트/
  2605091859 영상제목.mp4
  2605091859 영상제목.txt
  2605091859 영상제목_요약.txt

.txt 파일은 전체 스크립트입니다. 시간 라벨이나 내부 처리 문구 없이 본문만 들어갑니다.

_요약.txt 파일은 전체 스크립트를 바탕으로 만든 상세 요약입니다.
