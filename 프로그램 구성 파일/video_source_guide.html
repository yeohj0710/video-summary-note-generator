영상 가져오기 도움말

릴스, 유튜브 링크, 내 컴퓨터 영상/오디오 파일을 어떻게 넣어야 하는지 처음 쓰는 사람 기준으로 설명합니다.

1. 먼저 둘 중 하나만 고르세요

프로그램의 영상 가져오기 영역에는 두 가지 방식이 있습니다.

방식언제 쓰나요?무엇을 입력하나요?
링크로 가져오기 유튜브 영상이나 인스타그램 릴스를 인터넷 주소로 가져올 때 사용합니다. 영상 페이지 주소를 붙여넣습니다.
내 컴퓨터 파일 이미 PC에 저장된 녹화 파일, 강의 영상, 다운로드한 영상, mp3 녹음 파일을 사용할 때 선택합니다. 파일 선택 버튼으로 영상 또는 오디오 파일을 고릅니다.

링크와 파일을 동시에 넣는 것이 아닙니다. 위쪽 선택 버튼에서 현재 사용할 방식 하나만 고르면 됩니다.

2. 유튜브 링크 넣는 법

  1. 브라우저에서 유튜브 영상을 엽니다.
  2. 주소창의 주소를 복사합니다.
  3. 프로그램에서 링크로 가져오기를 선택합니다.
  4. 릴스 또는 유튜브 링크 입력칸에 붙여넣습니다.
  5. 노트 만들기를 누릅니다.

사용 가능한 예시

  • https://www.youtube.com/watch?v=...
  • https://youtu.be/...
  • https://www.youtube.com/shorts/...

비공개 영상, 멤버십 영상, 연령 제한 영상, 회사/학교 계정으로만 볼 수 있는 영상은 다운로드가 막힐 수 있습니다.

3. 인스타그램 릴스 링크 넣는 법

  1. 인스타그램에서 릴스 화면을 엽니다.
  2. 공유 또는 더보기 메뉴에서 링크를 복사합니다.
  3. 프로그램에서 링크로 가져오기를 선택합니다.
  4. 복사한 링크를 입력칸에 붙여넣습니다.
  5. 로그인이 필요한 릴스라면 브라우저 쿠키 사용을 켭니다.

쿠키 사용이 필요한 경우

  • 브라우저에서는 보이는데 프로그램에서는 다운로드가 실패할 때
  • 로그인해야 볼 수 있는 릴스일 때
  • 비공개 계정, 제한된 공개 범위, 연령 확인이 걸린 영상일 때

쿠키 사용을 켤 때는 실제로 로그인되어 있는 브라우저를 선택하세요. 예를 들어 Chrome에 로그인되어 있으면 chrome을 고르면 됩니다.

4. 내 컴퓨터 영상/오디오 파일 넣는 법

  1. 내 컴퓨터 파일을 선택합니다.
  2. 파일 선택 버튼을 누릅니다.
  3. 사용할 영상 또는 오디오 파일을 고릅니다.
  4. 노트 만들기를 누릅니다.

추천 파일 형식

  • mp4: 가장 추천합니다.
  • mov, m4v, 3gp, mkv, avi, webm: 휴대폰 녹화 파일을 포함해 대부분 처리 가능합니다.
  • mp3, wav, m4a, amr, caf, aiff, aac, flac: 오디오만 있어도 처리 가능합니다.

영상에 소리가 없거나 음성 트랙이 분리되어 있으면 전사가 실패할 수 있습니다. 영상 재생 시 사람 목소리가 들리는지 먼저 확인하세요.

5. 결과 파일은 어디에 생기나요?

작업이 끝나면 생성된 노트 폴더에 같은 이름의 파일들이 나란히 저장됩니다.

  • 영상제목.mp4 또는 회의녹음.mp3: 링크 영상은 다운로드되고, 내 컴퓨터 파일은 복사됩니다.
  • 영상제목.txt: 전체 스크립트입니다.
  • 영상제목_요약.txt: 전체 스크립트를 바탕으로 만든 요약입니다.

6. 자주 막히는 상황

상황해결 방법
링크를 넣었는데 다운로드가 안 됩니다. 주소가 https://로 시작하는지 확인하고, 브라우저에서 영상이 실제로 재생되는지 확인하세요.
릴스가 브라우저에서는 보이는데 프로그램에서는 안 됩니다. 브라우저 쿠키 사용을 켜고, 로그인된 브라우저를 선택하세요.
전사 중 실패합니다. 영상에 음성이 있는지 확인하세요. 너무 손상된 파일이면 다른 플레이어에서 다시 저장한 뒤 시도하는 것이 좋습니다.
결과 파일이 어디 있는지 모르겠습니다. 작업 완료 후 결과 폴더 열기 버튼을 누르세요.

7. 쉬운 사용 팁

  • 처음에는 짧은 영상으로 테스트해 보세요.
  • 유튜브 강의처럼 긴 영상은 처리 시간이 꽤 걸릴 수 있습니다.
  • 릴스처럼 짧은 영상은 요약보다 전체 스크립트가 더 유용할 때도 있습니다.
  • 다운로드가 반복해서 실패하면 영상을 브라우저에서 직접 다운로드하거나 녹화해서 내 컴퓨터 파일 방식으로 넣는 것이 가장 쉽습니다.
