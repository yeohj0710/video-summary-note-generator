OpenAI API Key 받는 법

영상·음성 요약 노트 생성기에서 전사와 요약을 만들려면 본인 OpenAI API Key가 필요합니다.

1. OpenAI Platform에 로그인

  1. platform.openai.com에 접속합니다.
  2. OpenAI 계정으로 로그인합니다. ChatGPT 계정과 같을 수 있지만, API 사용은 Platform에서 따로 관리됩니다.
  3. 처음 사용하는 계정이라면 화면 안내에 따라 조직 또는 프로젝트 설정을 완료합니다.

2. 결제 또는 크레딧 확인

  1. Billing Overview를 엽니다.
  2. API 사용 가능한 결제 수단 또는 크레딧이 있는지 확인합니다.
  3. 크레딧이 없거나 결제 설정이 없으면 전사나 요약 중 할당량 오류가 날 수 있습니다.

ChatGPT Plus, Pro 같은 구독과 OpenAI API 사용 요금은 별도로 취급될 수 있습니다. 이 프로그램은 OpenAI API를 사용합니다. 기본 설정 기준 10분 영상은 보통 수십 원 정도이며, 영상 길이와 말의 양에 따라 달라질 수 있습니다.

3. API Key 만들기

  1. API Keys 페이지를 엽니다.
  2. 사용할 프로젝트를 확인한 뒤 새 Secret Key를 만듭니다.
  3. 이름은 예를 들어 media-summary-note-generator처럼 알아보기 쉽게 입력합니다.
  4. 생성된 키를 복사합니다. 보통 sk-... 또는 sk-proj-... 형태입니다.

키는 생성 직후에만 전체 값을 볼 수 있습니다. 나중에 다시 전체 키를 확인할 수 없으니 안전한 곳에 보관하세요.

4. 프로그램에 입력

  1. 영상·음성 요약 노트 생성기를 실행합니다.
  2. OpenAI API Key 설정 영역의 API 키 입력칸에 복사한 키를 붙여넣습니다.
  3. 계속 같은 PC에서 쓸 예정이면 이 PC에 API 키 저장을 체크합니다.
  4. 설정 버튼을 누르면 입력칸이 잠기고, 이후에는 수정 버튼으로 다시 고칠 수 있습니다.

5. 오류가 날 때 확인할 것

  • API 키 앞뒤에 공백이 붙지 않았는지 확인합니다.
  • Billing에 결제 수단 또는 크레딧이 있는지 확인합니다.
  • 키를 삭제했거나 권한을 너무 제한했다면 새 키를 만들어 다시 입력합니다.
  • 키가 외부에 노출된 것 같으면 OpenAI Platform에서 즉시 삭제하고 새 키를 발급합니다.
