# 영상·음성 요약 노트 생성기

유튜브, 릴스, 내 컴퓨터의 영상/오디오 파일을 넣으면 원본 파일, 전체 스크립트 TXT, 상세 요약 TXT를 만들어 주는 Windows 프로그램입니다.

**developed by yeohj0710**

## 다운로드

1. 이 페이지 오른쪽 위의 초록색 `Code` 버튼을 누릅니다.
2. `Download ZIP`을 누릅니다.
3. 내려받은 ZIP 파일의 압축을 풉니다.
4. 폴더 안의 `영상·음성 요약 노트 생성기.exe`를 실행합니다.

![GitHub Download ZIP 위치](프로그램%20구성%20파일/github-download-zip.png)

## 사용 순서

1. `영상·음성 요약 노트 생성기.exe`를 실행합니다.
2. `링크로 가져오기` 또는 `내 컴퓨터 파일` 중 하나를 선택합니다.
3. 유튜브/릴스 링크를 붙여넣거나, 영상/오디오 파일을 선택합니다.
4. OpenAI API 키를 입력합니다.
5. 저장 폴더는 기본값 그대로 두면 `생성된 노트` 폴더에 저장됩니다.
6. `노트 만들기`를 누릅니다.
7. 완료되면 `결과 폴더 열기`를 눌러 결과 파일을 확인합니다.

## 결과 파일

작업이 끝나면 `생성된 노트` 폴더 안에 같은 이름의 파일 3개가 생깁니다.

```text
생성된 노트/
  2605091859 영상제목.mp4
  2605091859 영상제목.txt
  2605091859 영상제목_요약.txt
```

오디오 파일을 넣은 경우에도 같은 방식입니다.

```text
생성된 노트/
  2605091901 회의녹음.mp3
  2605091901 회의녹음.txt
  2605091901 회의녹음_요약.txt
```

`txt` 파일은 전체 스크립트이고, `_요약.txt` 파일은 전체 스크립트를 바탕으로 만든 상세 요약입니다.

## 지원 파일

- 링크: 유튜브, 인스타그램 릴스 등
- 영상: `mp4`, `mov`, `m4v`, `webm`, `mkv`, `avi`, `3gp` 등
- 오디오: `mp3`, `wav`, `m4a`, `amr`, `caf`, `aiff` 등

## 비용

OpenAI API 사용량에 따라 비용이 발생할 수 있습니다.

기본 설정 기준 10분 영상은 보통 수십 원 정도입니다. 영상 길이, 말의 양, 문장 정리 옵션에 따라 달라질 수 있습니다.

작업이 끝나면 프로그램의 진행 상황 로그에 한화 기준 예상 비용이 표시됩니다.

## 폴더 구성

| 항목 | 설명 |
| --- | --- |
| `영상·음성 요약 노트 생성기.exe` | 실행 파일입니다. |
| `사용설명서.html` | 압축을 푼 뒤 볼 수 있는 자세한 사용설명서입니다. |
| `생성된 노트` | 결과 파일이 저장되는 폴더입니다. |
| `프로그램 구성 파일` | 실행에 필요한 파일입니다. 삭제하거나 이름을 바꾸면 실행이 안 될 수 있습니다. |
