# 동영상 스크립트 TXT 생성기 사용설명서

영상을 넣으면 동영상 파일, 전체 스크립트 TXT, 상세 요약 TXT를 같은 이름으로 만들어 주는 Windows 프로그램입니다.

**developed by yeohj0710**

## 결과물

작업이 끝나면 `생성된 노트` 폴더 안에 파일 3개가 바로 생깁니다. 하위 폴더는 만들지 않습니다.

```text
생성된 노트/
  2605091859 영상제목.mp4
  2605091859 영상제목.txt
  2605091859 영상제목_요약.txt
  2605091907 다른영상.mp4
  2605091907 다른영상.txt
  2605091907 다른영상_요약.txt
```

파일명은 `YYMMDDHHMM 영상제목` 형식입니다. 예를 들어 2026년 5월 9일 18시 59분에 만든 파일은 `2605091859 영상제목.txt`처럼 저장됩니다.

## 사용 순서

1. `동영상 요약 노트 생성기.exe`를 실행합니다.
2. 유튜브/릴스 링크를 넣거나, 내 컴퓨터의 동영상 파일을 선택합니다.
3. OpenAI API 키를 입력합니다.
4. 저장 폴더가 `생성된 노트`인지 확인합니다.
5. `스크립트 만들기`를 누릅니다.
6. 완료되면 `결과 폴더 열기`를 눌러 영상, 전체 스크립트, 요약 파일을 확인합니다.

## 만들어지는 TXT

일반 TXT 파일에는 시간 라벨이나 내부 처리 문구를 넣지 않습니다. 전체 전사문만 들어갑니다.

문장은 읽기 편하도록 널널하게 줄바꿈됩니다.

```text
첫 번째 문장입니다.

두 번째 문장입니다.

세 번째 문장입니다.
```

`_요약.txt` 파일은 전체 스크립트를 마지막에 한 번 더 읽고 만든 상세 요약본입니다. 수치, 날짜, 금액, 조건, 인물/회사/제품명, 중요한 예외와 결론은 최대한 살리고 반복 표현만 줄이는 방식으로 생성합니다.

## 폴더 설명

| 항목 | 설명 |
| --- | --- |
| `동영상 요약 노트 생성기.exe` | 실행 파일입니다. |
| `사용설명서.html` | 일반 사용자가 보기 쉬운 HTML 설명서입니다. |
| `생성된 노트` | 완성된 동영상 파일, 전체 스크립트 TXT, 요약 TXT가 저장되는 폴더입니다. |
| `프로그램 구성 파일` | 실행에 필요한 구성 파일입니다. 삭제하거나 이름을 바꾸면 실행이 안 될 수 있습니다. |

## 참고

- 링크 영상은 다운로드된 동영상도 `생성된 노트` 폴더에 같이 저장됩니다.
- 내 컴퓨터 파일을 선택한 경우에도 원본 영상이 `생성된 노트` 폴더에 복사되고, 같은 이름의 TXT와 요약 TXT가 만들어집니다.
- 중간 작업 파일은 임시 폴더에서 처리되고 결과 폴더에는 남지 않습니다.
- OpenAI API 사용량에 따라 비용이 발생할 수 있습니다.
