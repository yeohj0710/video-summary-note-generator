## 1. 이 프로그램으로 할 수 있는 일

동영상 요약 노트 생성기는 영상 하나를 넣으면 아래 작업을 자동으로 처리합니다.

- 유튜브 또는 공개 릴스 링크에서 영상을 다운로드합니다.
- 내 컴퓨터에 있는 동영상 파일을 직접 불러옵니다.
- 영상의 음성을 한국어 텍스트로 전사합니다.
- 전사된 문장을 맞춤법, 띄어쓰기, 문장부호 중심으로 다듬습니다.
- 영상에서 중요한 내용이 나오는 시점을 고릅니다.
- 그 시점의 화면 이미지를 저장합니다.
- 요약, 주요 장면, 전체 전사문이 들어간 PDF와 텍스트 결과 파일을 만듭니다.

쉽게 말하면, 영상을 보고 직접 캡처하고 받아 적는 일을 프로그램이 대신 해 줍니다.

## 3. 가장 빠른 사용 순서

### 1. 프로그램 실행

압축을 푼 폴더에서 `동영상 요약 노트 생성기.exe`를 더블클릭합니다.

`프로그램 구성 파일` 폴더는 같은 위치에 그대로 두세요. 이 폴더는 프로그램 실행에 필요합니다.

### 2. 영상 가져오기

유튜브나 릴스 링크를 쓸 때는 `링크로 가져오기`를 선택하고 링크를 붙여 넣습니다.

내 컴퓨터 파일을 쓸 때는 `내 컴퓨터 파일`을 선택하고 `파일 선택`을 누릅니다.

### 3. API 키 입력

`API 키` 칸에 OpenAI API 키를 붙여 넣습니다.

개인 PC에서 매번 입력하기 귀찮다면 `이 PC에 API 키 저장`을 체크합니다.

### 4. 출력 폴더 확인

기본값은 프로그램 폴더 안의 `생성된 노트` 폴더입니다. 이 폴더는 처음 실행할 때 자동으로 만들어집니다. 바꾸고 싶으면 `폴더 선택`을 누릅니다.

### 5. 노트 만들기

`노트 만들기` 버튼을 누르고 기다립니다. 완료되면 `결과 폴더 열기` 버튼으로 결과를 확인합니다.

## 4. 화면별 설명

### 1. 영상 가져오기

링크 다운로드와 내 컴퓨터 파일 중 한 가지 방식만 고르는 곳입니다. 선택한 방식에 맞는 입력칸만 화면에 나타납니다.

- **링크로 가져오기:** 유튜브 또는 릴스 링크를 붙여 넣습니다.
- **내 컴퓨터 파일:** 컴퓨터에 저장된 동영상 파일을 선택합니다.
- **브라우저 쿠키 사용:** 링크로 가져올 때만 보이는 옵션입니다. 로그인해야 볼 수 있는 링크를 다운로드해야 할 때 사용합니다.

### 2. OpenAI 설정

AI가 음성을 듣고 텍스트를 만들기 위해 필요한 설정입니다.

- **API 키:** OpenAI API 키를 입력합니다.
- **이 PC에 API 키 저장:** 다음 실행 때도 키를 자동으로 불러옵니다.
- **전사 모델:** 음성을 텍스트로 바꾸는 모델입니다. 기본값 `gpt-4o-mini-transcribe`를 추천합니다.
- **문장/요약 모델:** 전사문을 다듬고 주요 장면을 고르는 모델입니다. 비용을 아끼려면 기본값 `gpt-5-nano`를 추천합니다.

### 3. 출력 설정

결과 파일을 어디에 만들지, 주요 장면을 몇 개 만들지 정합니다.

- **출력 폴더:** 결과가 저장될 위치입니다.
- **장면 수 자동 결정:** 영상 길이와 내용에 맞춰 AI가 장면 수를 정합니다.
- **직접 지정:** 자동 결정을 끄고 원하는 장면 수를 숫자로 입력합니다.
- **자동 최소/최대:** 자동으로 만들 때 너무 적거나 많아지지 않게 범위를 정합니다.

### 진행 상황

오른쪽 영역에는 현재 처리 단계가 표시됩니다. 다운로드, 음성 추출, 전사, 문장 정리, 주요 장면 선택, 이미지 추출 순서로 진행됩니다.

## 5. 유튜브 또는 릴스 링크 사용법

1. 프로그램에서 `링크로 가져오기`를 선택합니다.
2. 유튜브 또는 릴스 링크를 복사합니다.
3. `릴스 또는 유튜브 링크` 칸에 붙여 넣습니다.
4. API 키와 출력 폴더를 확인합니다.
5. `노트 만들기`를 누릅니다.

링크 영상은 분석 전에 프로그램 폴더 안의 `다운로드한 동영상` 폴더에 먼저 저장됩니다. 이후 저장된 원본 동영상을 기준으로 음성 전사와 요약 노트를 만듭니다.

유튜브 예시:

```text
https://www.youtube.com/watch?v=...
```

릴스가 실패할 때는 공개 영상이 아니거나 로그인이 필요한 영상일 수 있습니다. Chrome 또는 Edge에서 해당 영상을 볼 수 있는 상태로 로그인한 뒤, 프로그램의 `브라우저 쿠키 사용`을 켜고 다시 시도해 보세요.

## 6. 내 컴퓨터 동영상 사용법

1. 프로그램에서 `내 컴퓨터 파일`을 선택합니다.
2. `파일 선택` 버튼을 누릅니다.
3. 분석할 동영상 파일을 선택합니다.
4. API 키와 출력 폴더를 확인합니다.
5. `노트 만들기`를 누릅니다.

지원 가능성이 높은 파일 형식은 `mp4`, `mov`, `mkv`, `avi`, `webm`입니다.

오디오가 없는 영상은 영상에 음성 트랙이 없기 때문에 전사할 내용이 없어 실패할 수 있습니다.

## 7. OpenAI API 키 입력법

이 프로그램은 OpenAI API를 사용합니다. ChatGPT 로그인만으로는 동작하지 않고, API 키가 필요합니다.

### API 키를 넣는 위치

1. 프로그램 화면에서 `OpenAI 설정` 영역을 찾습니다.
2. `API 키` 칸에 키를 붙여 넣습니다.
3. 키는 보안을 위해 화면에서 점으로 표시됩니다.

### API 키 저장 옵션

- 개인 PC라면 `이 PC에 API 키 저장`을 체크하면 편합니다.
- 공용 PC, 회사 공용 노트북, 다른 사람이 쓰는 PC에서는 체크하지 않는 것이 좋습니다.
- 저장된 설정은 Windows 사용자 설정 폴더의 `%APPDATA%\VideoSummaryNoteGenerator\settings.json`에 저장됩니다.

키가 틀렸을 때 처리 중 OpenAI 오류가 나면 API 키가 잘못되었거나, 결제 설정이 없거나, 사용 한도가 부족할 수 있습니다.

## 8. 주요 장면 수 설정법

영상마다 좋은 장면 수가 다릅니다. 짧은 릴스는 몇 장면이면 충분하고, 긴 강의는 더 많은 장면이 필요합니다.

### 추천 설정

| 영상 종류 | 추천 설정 | 설명 |
| --- | --- | --- |
| 30초~2분 릴스 | 자동, 최소 4 / 최대 8 | 짧지만 장면 전환이 빠르므로 너무 적게 뽑지 않는 것이 좋습니다. |
| 5~20분 유튜브 영상 | 자동, 최소 6 / 최대 16 | 주제 전환과 예시를 적당히 잡기 좋습니다. |
| 1시간 강의 | 자동, 최소 10 / 최대 24 | 챕터처럼 핵심 흐름을 잡는 데 적합합니다. |
| 특정 개수가 꼭 필요할 때 | 자동 끄기, 직접 지정 | 보고서 형식이나 발표 자료에 맞춰 장면 수를 고정할 수 있습니다. |

처음 쓰는 경우 `장면 수 자동 결정`을 켜 둔 기본 설정을 추천합니다.

## 9. 결과 파일 보는 법

작업이 끝나면 `생성된 노트` 폴더 안에 새 폴더가 생깁니다. 폴더 이름은 보통 `날짜_시간_영상제목` 형태입니다.

링크로 가져온 원본 영상 파일은 `다운로드한 동영상` 폴더에서 따로 확인할 수 있습니다.

```text
생성된 노트
└─ 20260509_132000_영상제목
   ├─ 요약 노트.pdf
   ├─ 전체 스크립트.txt
   └─ 기타 파일
      ├─ summary.md
      ├─ summary.html
      ├─ metadata.json
      ├─ frames
      │  ├─ scene_01_00-00-12.jpg
      │  └─ scene_02_00-00-28.jpg
      └─ audio_chunks
```

| 파일 | 용도 |
| --- | --- |
| `요약 노트.pdf` | 공유, 보관, 출력에 가장 편한 최종 결과 파일입니다. 주요 화면과 스크립트가 함께 들어갑니다. |
| `전체 스크립트.txt` | 시간대별로 정리된 전체 전사문입니다. |
| `기타 파일` 폴더 | 필요할 때만 열어보는 보조 파일 폴더입니다. HTML, Markdown, 장면 이미지, metadata가 들어갑니다. |

## 10. Notion에 옮기는 법

일반 사용자에게 가장 쉬운 방법은 `요약 노트.pdf`를 Notion 페이지에 끌어다 놓는 방식입니다.

### 방법 A: PDF 넣기

1. 결과 폴더에서 `요약 노트.pdf`를 찾습니다.
2. Notion 페이지에 파일을 끌어다 놓습니다.
3. Notion에서 PDF를 미리보기 또는 첨부 파일로 확인합니다.

### 방법 B: Markdown 가져오기

1. Notion에서 새 페이지를 만듭니다.
2. 가져오기 또는 Import 메뉴를 사용합니다.
3. `기타 파일` 폴더 안의 `summary.md` 파일을 선택합니다.

Notion에 붙여 넣을 때 이미지가 자동으로 같이 들어가지 않는 경우가 있습니다. 그때는 `기타 파일\frames` 이미지를 Notion에 직접 끌어다 놓으면 됩니다.

## 11. 비용과 처리 시간

이 프로그램은 OpenAI API를 사용하기 때문에 영상 길이에 따라 비용과 시간이 달라집니다.

| 항목 | 영향 |
| --- | --- |
| 영상 길이 | 길수록 음성 전사 비용과 시간이 늘어납니다. |
| 말의 양 | 같은 10분 영상이라도 말이 많으면 처리할 텍스트가 많아집니다. |
| 주요 장면 수 | 이미지 추출 자체는 비용이 거의 없지만, 장면을 많이 고르면 결과 파일이 커집니다. |
| 모델 선택 | 고성능 모델을 쓰면 품질이 좋아질 수 있지만 비용이 더 커질 수 있습니다. |

처음에는 기본 모델과 자동 장면 수로 테스트한 뒤, 결과가 마음에 들면 긴 영상에 사용하세요.

## 12. 문제가 생겼을 때

### 프로그램이 안 켜져요

- 압축 파일 안에서 바로 실행하지 말고, 먼저 압축을 완전히 푼 뒤 실행해 보세요.
- Windows 보안 경고가 뜨면, 신뢰할 수 있는 배포 파일인지 확인한 뒤 실행하세요.

### 유튜브 또는 릴스 다운로드가 실패해요

- 링크가 공개 영상인지 확인하세요.
- 브라우저에서는 보이지만 프로그램에서 실패하면 `브라우저 쿠키 사용`을 켜고 Chrome 또는 Edge를 선택하세요.
- 비공개 영상, 지역 제한 영상, 로그인 필수 영상은 실패할 수 있습니다.

### OpenAI 오류가 나요

- API 키를 다시 복사해서 붙여 넣어 보세요.
- OpenAI API 결제 설정이 되어 있는지 확인하세요.
- 계정 사용 한도 또는 결제 한도에 걸리지 않았는지 확인하세요.

### 전사 결과가 이상해요

- 영상 음질이 낮거나 배경음이 크면 전사 품질이 떨어질 수 있습니다.
- 여러 사람이 동시에 말하면 문장이 어색해질 수 있습니다.
- 한국어와 외국어가 섞인 영상은 일부 단어가 다르게 적힐 수 있습니다.

### 결과 파일을 못 찾겠어요

- 프로그램에서 `결과 폴더 열기` 버튼을 누르세요.
- 직접 찾을 때는 프로그램 폴더 안의 `생성된 노트` 폴더를 확인하세요.
- 출력 폴더를 바꿨다면, 바꾼 위치에 결과가 저장됩니다.
